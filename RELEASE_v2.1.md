# FlowTick V2.1

V2.1 聚焦体验细节修复与统计功能增强，新增会话持久化、趋势图表等实用功能。

## 新增功能

- **终止会话按钮** — 首页新增 ■ 终止按钮，提前结束专注并记录已专注时间（区别于重置的清空不记录）
- **会话持久化** — 计时状态（运行中 / 暂停）自动保存到 session.json，关闭程序后重新打开恢复剩余时间和已专注时长，运行中退出会自动修正时间漂移
- **事项统计** — 统计页新增"事项统计"区块，按事项聚合专注次数和总时长，按时长降序展示前 10
- **近 14 天趋势折线图** — 统计页新增趋势可视化，展示最近 14 天每日专注时长变化，今日高亮，窗口拉伸自适应
- **文件夹可调宽度** — 笔记页面文件夹面板与笔记区之间新增可拖动分隔线（60~300px）
- **文件夹名省略号** — 长文件夹名超出面板宽度时自动截断并显示省略号，悬停显示完整名称
- **对话框 Enter 确认** — 专注设置、笔记、文件夹三个对话框均支持 Enter 键确认（笔记内容框中 Enter 仍为换行）

## 改进

- Tooltip 样式从黄色底色改为白色卡片，与整体 UI 风格统一
- 统计页面图表（柱状图、折线图）宽度随窗口拉伸自适应
- 趋势图日期标签增大字体（7→9），调整显示间距避免重叠
- 笔记文件夹标签字体增大（10→11），底部按钮字体增大（9→10）
- 事件时间格式包含年份（YYYY-MM-DD HH:MM），解决跨年数据统计错误
- 统计页删除事件后，总体统计（累计次数、时长、天数）同步更新
- 事件页删除事件后，统计汇总数据同步更新

## 修复

- 修复跨年事件删除和统计中的年份硬编码问题
- 修复笔记页面 tooltip 在点击文件夹后残留不消失
- 修复 `_is_click_on_focusable` 遇到字符串 widget 时崩溃
- 修复切换到其他页面再回统计页时图表不撑满窗口
- 修复趋势图日期标签相邻两天重叠显示
- 修复 notes.json 中混入事件数据导致笔记页面崩溃

## 安装

### 从源码运行

```bash
git clone https://github.com/ayuan-01/FlowTick.git
cd FlowTick
pip install pystray Pillow
python main.py
```

### 打包为 exe

```bash
pip install pyinstaller pystray Pillow
pyinstaller FlowTick.spec
```

生成的可执行文件位于 `dist/FlowTick.exe`。
