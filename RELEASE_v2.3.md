# FlowTick V2.3

V2.3 聚焦交互体验提升，新增圆弧进度条、图表悬停、快捷键、删除确认等多项改进。

## 新增功能

- **圆弧进度条** — 计时器从静态圆角矩形升级为圆弧进度条，实时显示当前段落剩余时间
- **每日目标进度** — 设置页新增每日番茄目标（默认 8 个），首页按钮行下方以小圆点展示今日完成进度
- **图表悬停数值** — 柱状图和趋势折线图支持鼠标悬停，弹出 tooltip 显示具体日期和分钟数
- **键盘快捷键扩展** — S（跳过）、Escape（终止）、T（添加待办）、1-5（切换页面）
- **删除确认弹窗** — 删除事件、笔记、待办、文件夹、清除已完成均弹出二次确认
- **托盘菜单增强** — 新增"开始/暂停"快捷操作，无需打开主窗口即可控制计时

## 改进

- **事件页日期分组** — 事件列表按日期分组显示（今天 / 昨天 / MM 月 DD 日），时间仅显示时分
- **状态标签独立** — 新增 status_label 显示会话进度，topic_label 回归显示事项名称
- **笔记拖放高亮** — 拖动笔记时目标文件夹高亮显示，松手后恢复
- **设置页分区** — 设置面板按"通用"、"计时节奏"、"闲置检测"分组，结构更清晰
- **确认弹窗优化** — 白色卡片横向填满对话框，长文本自动换行，比例更协调

## 修复

- **修复计时器文字裁切** — 圆弧进度条与文字背景重叠导致右侧文字被截断，重新设计为居中圆弧 + 中心遮罩布局
- **修复确认弹窗比例** — 内部白色卡片不随对话框拉伸，视觉不协调，添加 fill=X 填充

## 安装

### 从源码运行

```bash
git clone https://github.com/ayuan-01/FlowTick.git
cd FlowTick
pip install pystray Pillow
python main.py
```

### 打包为 exe

```bash
pip install pyinstaller pystray Pillow
pyinstaller FlowTick.spec
```

生成的可执行文件位于 `dist/FlowTick.exe`。
